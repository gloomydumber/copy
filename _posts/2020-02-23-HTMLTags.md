## Text Blocking Tags

### hn tags

제목을 나타내는 태그

```html
<h1>제목을 나타내는 태그</h1>
```

### p tag

단락을 나타내는 태그

```html
<p>단락을 나타내는 태그</p>
```

### br tag

줄 바꾸기

### hr tag

수평 줄 넣기

```html
<hr />
```

### blockquote tag

인용문 넣기

```html
<blockquote>인용 내용</blockquote>
```

### pre tag

입력하는 그대로 화면에 표시(공백 무시 하지 않음), 접근성 고려 필요(낭독기)

```html
<pre> 텍스트 </pre>
```

## Text in-Line Level Tags

### strong, b tags

텍스트 굵게 강조, 표시

```html
<strong> 텍스트 </strong> <b> 텍스트 </b>
```

### em, i tags

이랠릭, 비스듬히 표시

```html
<i> 텍스트 </i> <em> 텍스트 </em>
```

### q tag

인용 내용 표시

```html
<q> 인용 내용 </q>
```

### mark tag

형광펜 효과

```html
<mark> 텍스트 </mark>
```

### span tag

줄바꿈 없이 영역 묶기

```html
<span> 내용 </span>
```

### ruby tag

동아시아 글자 표시

```html
<ruby> 내용 <rt> 주석 </rt> </ruby>
```

### etc

```html
<abbr> 약자 </abbr>
<cite> 참고내용 </cite>
<code> 소스코드 </code>
<kbd> 키보드 입력 </kbd>
<small> 부가 정보 </small>
<sub> 아래첨자 </sub>
<sup> 위첨자 </sup>
<s> 취소선 </s>
<u> 밑줄 </u>
```

## List making Tags

### ul, li tags

순서가 필요하지 않은 목록(unorederd list)

```html
<ul>
  <li>내용</li>
  <li>내용</li>
</ul>
```

### ol, li tags

순서가 필요한 목록 (ordered list)
type 속성으로 1, a, A, i, I

```html
<ol>
  <li>내용</li>
  <li>내용</li>
</ol>

<ol type="a">
  <li>내용</li>
</ol>
```

### dl, dt, dd tags

제목과 설명이 한 쌍인 설명 목록(description list)

```html
<dl>
  <dt>제목</dt>
  <dd>설명</dd>
</dl>
```
