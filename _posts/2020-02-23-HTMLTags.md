## Text Blocking Tags

### hn tags

제목을 나타내는 태그

```html
<h1>제목을 나타내는 태그</h1>
```

### p tag

단락을 나타내는 태그

```html
<p>단락을 나타내는 태그</p>
```

### br tag

줄 바꾸기

### hr tag

수평 줄 넣기

```html
<hr />
```

### blockquote tag

인용문 넣기

```html
<blockquote>인용 내용</blockquote>
```

### pre tag

입력하는 그대로 화면에 표시(공백 무시 하지 않음), 접근성 고려 필요(낭독기)

```html
<pre> 텍스트 </pre>
```

## Text in-Line Level Tags

### strong, b tags

텍스트 굵게 강조, 표시

```html
<strong> 텍스트 </strong> <b> 텍스트 </b>
```

### em, i tags

이랠릭, 비스듬히 표시

```html
<i> 텍스트 </i> <em> 텍스트 </em>
```

### q tag

인용 내용 표시

```html
<q> 인용 내용 </q>
```

### mark tag

형광펜 효과

```html
<mark> 텍스트 </mark>
```

### span tag

줄바꿈 없이 영역 묶기

```html
<span> 내용 </span>
```

### ruby tag

동아시아 글자 표시

```html
<ruby> 내용 <rt> 주석 </rt> </ruby>
```

### etc

```html
<abbr> 약자 </abbr>
<cite> 참고내용 </cite>
<code> 소스코드 </code>
<kbd> 키보드 입력 </kbd>
<small> 부가 정보 </small>
<sub> 아래첨자 </sub>
<sup> 위첨자 </sup>
<s> 취소선 </s>
<u> 밑줄 </u>
```

## List making Tags

### ul, li tags

순서가 필요하지 않은 목록(unorederd list)

```html
<ul>
  <li>내용</li>
  <li>내용</li>
</ul>
```

### ol, li tags

순서가 필요한 목록 (ordered list)
type 속성으로 1, a, A, i, I

```html
<ol>
  <li>내용</li>
  <li>내용</li>
</ol>

<ol type="a">
  <li>내용</li>
</ol>
```

### dl, dt, dd tags

제목과 설명이 한 쌍인 설명 목록(description list)

```html
<dl>
  <dt>제목</dt>
  <dd>설명</dd>
</dl>
```

## table making tags

### table, tr, td, th tags

기본적인 표 만들기

```html
<table>
  <tr>
    <th>제목</th>
    <td>1행 2열</td>
    <td>1행 3열</td>
  </tr>
  <tr>
    <th>제목</th>
    <td>2행 2열</td>
    <td>2행 3열</td>
  </tr>
</table>
```

### colspan, rowspan Attributes

셀 합병

```html
<td colspan="합칠 셀의 개수">내용</td>
<th colspan="합칠 셀의 개수">내용</th>
<td rowpsan="합칠 셀의 개수">내용</td>
<th rowspan="합칠 셀의 개수">내용</th>
```

### caption, figcaption tags

표에 제목 붙이기

```html
<table border="1">
  <caption>
    <strong>표 이름, 표 제목</strong>
    <p>표에 관한 설명</p>
  </caption>
  <tr>
    <th>내용</th>
    <th>내용</th>
  </tr>
</table>

<figure>
  <figcaption>
    <p>표 <b>이름</b></p>
  </figcaption>
  <table border="1">
    <tr>
      <th>내용</th>
      <th>내용</th>
    </tr>
  </table>
</figure>
```

### thead, tbody, tfoot tags

표 구조 정의하기

```html
<table>
  <thead>
    <tr>
      내용
    </tr>
  </thead>
  <tbody>
    <tr>
      내용
    </tr>
  </tbody>
  <tfoot>
    <tr>
      내용
    </tr>
  </tfoot>
</table>
```

### col, colgroup tags

여러 열 묶어 스타일 지정하기

```html
<table>
  <colgroup>
    <col span="2" style="background-color:blue;" />
  </colgroup>
  <tr>
    <td>내용</td>
    <td>내용</td>
  </tr>
  <tr>
    <td>내용</td>
    <td>내용</td>
  </tr>
</table>
```

## image tags

### img tag

이미지 삽입

```html
<img src="경로" [속성="값" ] />
```

### alt attribute

이미지를 설명해 주는 대체 텍스트

```html
<img src="picture1.jpg" alt="그림1" />
```

### width, height attributes

이미지 크기 조정하기

```html
<img src="picture2.jpg" width="250" height="90" alt="그림2" />
```

### figure, figcaption tags

이미지에 설명 글 붙이기
figure tag는 한 단위가 되는 요소를 묶음

```html
<figure>
  <img src="picture3.jpg" alt="그림3" />
  <figcaption>설명 글</figcaption>
</figure>
```

## link tags

### a tag, href attribute

링크 만들기

a tag's attributes : href, target, download, rel, hreflang, type

```html
<a href="링크 주소" [속성="속성 값" ]> 텍스트 </a>
<a href="링크 주소" [속성="속성 값" ]> <img src="이미지 경로" /> </a>
```

### target attribute

새 탭에서 링크 열기

속성 값
\_blank : 링크 내용이 새 창, 새 탭에서 열림
\_self : target 속성의 기본 값, 같은 화면에서 열림
\_parent : 프레임을 사용 했을 때, 링크 내용을 부모 프레임에 표시
\_top : 프레임을 사용 했을 때, 링크 내용을 전체화면에 표시

```html
<a href="http://naver.com" target="_blank"> 네이버 </a>
```

### anchor

한 페이지 안에서 점프하는 기능 (# 이용)

```html
<태그 id="앵커 이름"> 텍스트 또는 이미지 </태그>
<a href="#앵커 이름"> 텍스트 또는 이미지 </a>
```

### map, area tags, usemap attribute

이미지 맵 지정하기

```html
<map name="맵이름">
  <area />
  <area />
</map>
<img src="이미지 경로" usemap="#맵이름" />
```
